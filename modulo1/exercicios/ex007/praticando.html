Aplicando as printipais formatações

Negrito

Para negritar um texto em HTM4 usamos a tag b que atualmente é obsoleta

Já no caso do HTML5 negritamos com a tag strong

Itálico

Podemos colocar a palavra em itálico com a tag i obsoleta do HTML4

Mas o certo é usar a tag em do HTML5 para fazer isso

Marcar textos

Usamos a tag mark para isso

Devemos usar o CSS3 para mudar a cor dessa marca

Textos grandes e pequenos

A tag big atualmente obsoleta põe o texto grande, mas a tag small faz a frase pequena

Deletar ou apagar

Apenas usamos a tag del para excluir um texto.

Sublinhar

A tag u obsoleta do HTM4 sublinha o texto, mas o certo é usar a tag ins do HTML5 para sublinhar.

Texto elevado ou sobrescrito

Usamos a tag sup por exemplo para escrever x2.

Texto subscrito ou índice

Usamos a tag sub para escrever x2.
